# hearmetalk

## Frontend Project setup
### モジュールのインストール
```
cd ./frontend
npm install
```

### Compiles and hot-reloads for development
```
npm run dev
```

### Compiles and minifies for production
```
npm run build
npm run serve
```

### Lints and fixes files
```
npm run lint
```
